# 🏗 Architecture — SwiftChat

## High-Level Architecture

Frontend (React)
↓
Backend (Node.js)
↓
AI Services (FastAPI)
↓
Databases (PostgreSQL, MongoDB, Redis)

---

## Layers

### Frontend

* UI rendering
* API calls

### Backend

* Business logic
* Auth
* API gateway

### AI Services

* Caption generation
* Emotion search
* Chatbot

### Data Layer

* PostgreSQL
* MongoDB
* Redis

---

## Architecture Type

* Microservices (AI separated)
* REST-based communication
* Scalable & modular
